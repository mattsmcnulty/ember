# Media for the README

Drop product captures here, then uncomment the `<p align="center">…</p>` block near the top of the
root [README](../../README.md):

- **`live-activity.gif`** — the Lock-Screen Live Activity with the temperature ticking up (and/or
  the Dynamic Island). Trim to under ~5 s and keep it under ~3 MB.
- **`control.png`** — the Control screen. Keep it under ~500 KB.

Capture via the iOS Simulator or a device screen recording (QuickTime). If a file exceeds those
sizes, attach it to a GitHub Release and hotlink it instead of committing it (keeps the repo lean).
